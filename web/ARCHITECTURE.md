# Smart Ledger UI - Web Interface

## 一、设计目标

参考 CodeObservatory 的 UI 设计语言，为 Smart Ledger 打造一个现代化的 Web 界面。

## 二、设计语言

### 2.1 色彩系统（深色主题为主）

```
--cosmic-bg: #020010          深空背景
--cosmic-surface: #080420     卡片表面
--cosmic-sidebar: #040218     侧边栏
--cosmic-border: #1a1050      边框
--cosmic-text: #e8e0f0        主文字
--cosmic-text-dim: #8070a0    次要文字
--cosmic-accent: #8880ff      主强调色（紫）
--cosmic-glow: #6460ff        发光效果

辅助色：
--success: #50c878            收入/正向
--warning: #ffb432            警告
--danger: #ff5050             超支/负向
--info: #06b6d4               信息（青色）
```

### 2.2 字体

- 标题：Georgia, "Times New Roman", serif（衬线体，大号数字）
- 正文：SF Pro Display, system-ui, sans-serif
- 代码/金额：SF Mono, monospace

### 2.3 组件风格

- 卡片：圆角 8px，1px 边框，微妙的背景色
- 导航：左侧 200px 侧边栏，可折叠
- 统计卡片：3 列网格，serif 大数字，小写标签
- 表格：紧凑行高，hover 高亮
- 图表：文本进度条（暂不用 chart 库）
- 动画：fade-in + stagger

## 三、技术栈

| 组件 | 选型 |
|------|------|
| 框架 | React 18 + TypeScript |
| 构建 | Vite 6 |
| 样式 | Tailwind CSS 4 + CSS Variables |
| 图标 | Lucide React |
| 动画 | Framer Motion |
| 路由 | React Router v6 |
| 后端 | 连接 Smart Ledger Python API (Flask) |

## 六、API 设计

Smart Ledger 需要一个 Flask API 层供前端调用：

```
GET  /api/transactions?month=2026-06     # 获取交易列表
POST /api/transactions                   # 添加交易（自然语言）
GET  /api/transactions/summary           # 月度汇总
GET  /api/budgets?month=2026-06          # 获取预算列表
POST /api/budgets                        # 设置预算
GET  /api/report?month=2026-06           # 生成月度报告
GET  /api/categories                     # 获取分类列表
GET  /api/currencies                     # 获取汇率
```
